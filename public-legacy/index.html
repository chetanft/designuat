üîå API Reference

Use these endpoints to integrate the comparison tool into your workflow:

Supported Figma URL Formats:

Design Files:
https://www.figma.com/design/[fileId]/[fileName]?node-id=[nodeId]

File View:
https://www.figma.com/file/[fileId]/[fileName]?node-id=[nodeId]

Prototype:
https://www.figma.com/proto/[fileId]/[fileName]?node-id=[nodeId]

Note: The node-id parameter is optional. If omitted, the entire file will be analyzed.

Single Comparison

POST /api/compare

Batch Comparison

POST /api/compare/batch

List Reports

GET /api/reports

Health Check

GET /api/health

Authentication Examples:

Basic Request Format:
curl -X POST http://localhost:[PORT]/api/compare \
-H "Content-Type: application/json" \
-d '{
"figmaUrl": "string (required)",
"webUrl": "string (required)",
"webSelector": "string (optional)",
"includeVisual": "boolean (optional)"
}'
With Authentication:
curl -X POST http://localhost:[PORT]/api/compare \
-H "Content-Type: application/json" \
-d '{
"figmaUrl": "string",
"webUrl": "string",
"authentication": {
"type": "credentials|cookies|headers|manual",
// Additional fields based on type
}
}'
Batch Processing:
curl -X POST http://localhost:[PORT]/api/compare/batch \
-H "Content-Type: application/json" \
-d '{
"comparisons": [
// Array of comparison objects
]
}'
